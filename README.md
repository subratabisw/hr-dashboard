# 👨‍💼 HR Analytics Dashboard

An interactive **HR Analytics Dashboard** built using **Microsoft Power BI** to monitor employee data, workforce insights, and HR performance metrics. The dashboard helps HR professionals analyze employee information, attrition, income, job satisfaction, and workforce demographics.

---

## 📊 Dashboard Preview

![HR Dashboard](dashboard.png)

---

## 🚀 Key Features

- 👥 Total Employee Count
- 📉 Total Attrition Status
- 💰 Average Monthly Income
- ⭐ Average Job Satisfaction
- ⚖️ Average Work-Life Balance
- 🎂 Employee Age Analysis
- 🏢 Department-wise Employee Distribution
- 👨‍💼 Gender Filter
- 💍 Marital Status Filter
- ⏰ Overtime Analysis
- 📈 Interactive Charts & Slicers
- 🏠 Navigation Button

---

## 📈 Dashboard Insights

This dashboard provides valuable HR insights including:

- Employee demographics
- Department-wise workforce distribution
- Attrition overview
- Salary analysis
- Job involvement by hourly rate
- Monthly income by performance rating
- Standard hours across salary slabs
- Interactive filtering for deeper analysis
